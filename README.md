# Task 4 – Marketing Campaign Analysis

## Overview
This project was completed as part of my Data Analytics Internship at SkillCraft Technology.

The objective of this task was to perform Exploratory Data Analysis (EDA) on a marketing campaign dataset to understand customer purchasing behavior, evaluate campaign performance, and provide business recommendations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data Cleaning and Preparation
- Handling Missing Values
- Customer Spending Analysis
- Marketing Channel Analysis
- Campaign Performance Evaluation
- Customer Response Analysis
- Business Insights and Recommendations

## Key Insights
- Wine products generated the highest customer spending.
- Store purchases were the most preferred purchasing channel.
- Campaign 4 achieved the highest customer acceptance.
- Customers with higher educational qualifications showed higher average spending.
- Personalized marketing strategies can improve campaign response rates.
